// Additional Buttons:

📝 - Sticky Note

- Show/Hide the Sticky Note.

- Click and drag the top of the Sticky Note to move around the page.

- The Sticky Note will not be included in the generated note, and will be cleared on refresh, or by selecting the 'Clear Notes' button.

🌓 - Theme switcher

- Toggle between different themes.

🔩 - Compact Mode

- Toggle Compact/Normal Size Modes.

📗 - Tier 1 Comments

- Show/Hide Tier 1 Hints and Suggestions for Tier 2 agents.

🧮 - EOI Calculator

- Use to calculate points completed during an Evidence of Identity check

💻 - Embedded

- Open a compact version of an embedded webpage (update note before use)

// How to Use:

1. Select the Copy Note button or use the hotkey [Alt + C] on your keyboard to copy the generated note to your clip board.

2. Navigate to the CRM note field and use the hotkey [Ctrl + V] to paste the generated notes.

3. Once notes have been transferred, use the Clear Notes button or use the hotkey [Alt + Z] to clear the selections made.

Hotkeys You can use [Alt + 1-6] to quickly toggle the additional buttons

- e.g. [Alt + 1] will quickly open/close the EOI Calculator
