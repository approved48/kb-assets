HTML2Docx CONVERSION TOOL v0.1

Will convert files listed as HTML into word document form

Things to consider: 
- Screenshots/Images will not be kept as it does not download
- Any information that is embedded from other sources will be missing as it purely pulls from the HTML provided

On first use you will need to edit the 'Run-html2docx-conversion' via notepad to add the conversion folder path

Steps:
1. Open Notepad and paste the HTML code
2. Select 'Save as' from the file menu
3. Set location as the 'Conersion' folder
4. Set the filename as "articlename.html"
5. Set 'Save as type' to "all files"
6. Select save
7. Double-click to run the 'Run-html2docx-conversion' script
8. Done!
